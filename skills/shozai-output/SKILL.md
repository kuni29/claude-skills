# 商材情報出力プロンプト

指定されたURLのページから、サービスの概要を以下のフォーマットでまとめてください。
引用元は不要です。コードブロックで出力してください。

## 出力フォーマット

```
ターゲット企業セグメント:
ターゲット人物セグメント:
商材名:
商材の機能：
商材が解決する課題:
商材による顧客価値（定量）:
商材による顧客価値（定性）:
商材による顧客価値（コンセプト）:
競合製品の特徴:
自社商材の特徴:
商材導入企業例:
CV動線設計:
アプローチ先人物が関心あるテーマ:
関心テーマに関するプレゼント情報:
アプローチ先人物の部署や役職:
アプローチ先人物のミッション:
アプローチ先企業の実現したいこと:
```

## 参考例（アポドリ）

```
ターゲット企業セグメント: 従業員51名以上の企業（特に成長期で営業活動に投資したい企業）
ターゲット人物セグメント: 経営層、営業部門、マーケティング部門、DX推進部門 / 役員・CxO、部長・本部長、課長クラス

商材名: 営業AIエージェント「アポドリ」
商材の機能：
- アポイント獲得業務の完全遂行（リスト作成、アプローチ実行、活動管理・分析）
- 企業ごとの営業プロセスへのカスタマイズ、AIの継続学習による品質向上
- AIと営業エキスパートのハイブリッドによる「勝ちパターン開発」

商材が解決する課題:
- 営業職の採用難・人材不足による営業リソースの不足
- 従来の営業DX（ツール導入）の行き詰まり、成果が出ない
- 負荷の高い営業アプローチ業務（商談獲得）にかかる工数
- 営業ノウハウの属人化

商材による顧客価値（定量）:
- 商談獲得率や成約率の向上、新規売上への貢献
- 営業リソース（人）の採用・育成・マネジメントコストの削減
- 24時間365日の営業活動実行によるアクション量の担保

商材による顧客価値（定性）:
- 営業パーソンが商談などの本質的な業務に集中できる環境の実現
- 属人化していた営業ノウハウを「勝ちパターン」として組織に資産化
- 営業組織全体の営業力向上

商材による顧客価値（コンセプト）: 人を増やさず、商談獲得から売上最大化まで成果をあげる

競合製品の特徴:
- 海外製品：AIによる営業支援（電話対応含む）の自動化
- 国内他社製品：既存SFA/CRM基盤上での管理効率化、データ分析によるアプローチ最適化

自社商材の特徴:
- AIがインサイドセールス業務（商談獲得）を自律的に遂行する
- AIと営業エキスパートのハイブリッド体制で「勝ちパターン」を開発
- 企業の個別プロセスに合わせてカスタマイズ可能

商材導入企業例: 導入事例ページ参照
CV動線設計: お問い合わせフォーム（資料請求・サービス紹介・個別商談）
アプローチ先人物が関心あるテーマ: 営業AIエージェントの可能性、営業DXの推進、営業職の採用難対策
関心テーマに関するプレゼント情報: サービス紹介資料、ウェビナーへの招待
アプローチ先人物の部署や役職: 経営、営業、マーケティング、DX関連 / 役員・CxO、部長・本部長、課長
アプローチ先人物のミッション: 営業成果の最大化、新規商談の安定的な創出
アプローチ先企業の実現したいこと: 人を採用せずに営業のアクション量を増やし、持続的な売上成長を実現する
```
