# 商談後メール自動作成

## 出力形式の厳守事項

- 必ずコードブロック(```)で囲んで出力する
- 段落間の空白行は1行のみ
- 長い文章（3行以上）は適切な場所で改行する
- 宛先は「企業名\n担当者名様」の形式
- 要望に応じて謝罪文やPDF添付の文言を追加する

**改行例：**
```
× 「改めまして、貴重なお時間をいただき誠にありがとうございました。介護業界の現場で…」（1行）
○ 「改めまして、貴重なお時間をいただき誠にありがとうございました。
介護業界の現場で奮闘されている職員の皆様のお役に立てるよう、精一杯サポートさせていただきます。
今後ともどうぞよろしくお願いいたします。」
```
