# Nano Banana：顔パーツ差分・プロンプト参考

**目的**: 同じ被写体条件（正面固定）のまま、**目・鼻・口**の各パーツだけを変えたとき、モデルがどこまで意図通りに分離して描けるかを確認したプロンプト集。

キャラクター設計時、**「こういう目（鼻・口）にしたい」**という具体的なプロンプトの書き方を探すための**リファレンス（参考資料）**として活用する。

---

## 収録ファイル一覧

| ファイル | 内容 |
| --- | --- |
| **[`eye_variations.md`](eye_variations.md)** | **目周り**（形・二重・目頭・まつ毛・涙袋・虹彩の大きさと色）の差分プロンプト集 |
| **[`nose_variations.md`](nose_variations.md)** | **鼻**（高さ・鼻筋・鼻先・小鼻・忘れ鼻）の差分プロンプト集 |
| **[`mouth_variations.md`](mouth_variations.md)** | **口周り**（唇の厚み・形・口角・人中・M字リップ・質感）の差分プロンプト集 |

---

## 活用方法（キャラクター設計時）

1. `reamero-character-prompt` スキルでキャラクターの「顔（Face）」を設計する際、より詳細なパーツ指定が必要な場合にこのフォルダを参照する。
2. 例えば「アーモンドアイで幅広二重にしたい」場合、`eye_variations.md` の中から近いプロンプトを探し、その英語表現をコピーしてキャラクターのプロンプトに組み込む。
3. **注意**: AIは顔パーツを細かく指定しすぎると解釈が破綻しやすいため、**特徴的なパーツ指定は1キャラクターにつき1〜3個程度**に留めること（引き算の設計）。
